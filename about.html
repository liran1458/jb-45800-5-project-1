About This Project

This Expense Manager system allows users to record, edit, filter, and analyze their expenses. The project includes a home page for managing entries, a filters page for advanced searching, and a graphs page that displays visual insights such as pie charts and bar charts. All data is stored locally in the browser using LocalStorage.
